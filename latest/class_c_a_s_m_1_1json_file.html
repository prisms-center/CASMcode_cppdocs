Inheritance diagram for CASM::jsonFile:

Detailed Description

Slightly more compact construction of jsonParser from file.

// enables using:
jsonFile json_a("path/to/file.json");
// instead of:
jsonParser json_b(fs::path("path/to/file.json"));
