Detailed Description

An abstract base class for sampling and storing data observations.

  • Derived classes "know" how to sample a particular property via implementation of 'virtual void sample(const MonteCarlo &mc, const MonteCounter &counter) = 0;'
  • Optionally may require and check for convergence to some level of precision given a particular confidence level
