Detailed Description

Specifies the composition on one or more equivalent sublattices

Example, specify sublattices 0 and 1 are "A":0.5, "B":0.5 with:

SublatComp sublat_comp { {0, 1}, {{"A", 0.5}, {"B", 0.5}} };
